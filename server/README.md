### Robonomics Node

We are using the server to feed data to robonomics node's datalog pallet.

```
Get robonomics node from https://github.com/airalab/robonomics/releases/tag/v2.6.0
Unzip the node
chmod +x ./robonomics
./robonomics --dev --tmp

```

### Installation
```
yarn
```

### Run the node
```
node server.js
```
